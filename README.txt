================================================================================
                            VULNERABILITY SCANNER TOOL
                                 README.txt
================================================================================

الأداة: ماسح الثغرات الأمنية المتقدم
النظام المستهدف: كالي لينكس (Kali Linux)
اللغة: Go (Golang)
الترخيص: مفتوح المصدر

================================================================================
                            متطلبات النظام
================================================================================

1. نظام تشغيل: كالي لينكس (أو أي توزيعة لينكس أخرى)
2. Go: الإصدار 1.19 أو أحدث
3. صلاحيات root أو sudo (للمسح الشبكي المتقدم)
4. اتصال إنترنت نشط (للحصول على قواعد بيانات CVE المحدثة)

================================================================================
                            خطوات التثبيت
================================================================================

1. تحديث النظام:
   sudo apt update && sudo apt upgrade -y

2. تثبيت Go (إذا لم يكن مثبتًا):
   sudo apt install golang-go -y
   
   أو تحميله من الموقع الرسمي:
   wget https://golang.org/dl/go1.21.0.linux-amd64.tar.gz
   sudo tar -C /usr/local -xzf go1.21.0.linux-amd64.tar.gz
   echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
   source ~/.bashrc

3. استنساخ المشروع:
   git clone [repository-url]
   cd vulnerability-scanner

4. تثبيت التبعيات:
   go mod download
   go mod tidy

5. بناء الأداة:
   go build -o vulnscanner main.go

6. منح الصلاحيات التنفيذية:
   chmod +x vulnscanner

================================================================================
                            استخدام الأداة
================================================================================

1. المسح الأساسي لعنوان IP:
   sudo ./vulnscanner scan 192.168.1.1

2. المسح المتقدم بنطاق عناوين:
   sudo ./vulnscanner scan 192.168.1.0/24

3. مسح محدد المنافذ:
   sudo ./vulnscanner scan 192.168.1.1 --ports 80,443,22,21

4. مسح شامل مع جميع الميزات:
   sudo ./vulnscanner scan 192.168.1.1 --ports 1-65535 --threads 100 --timeout 5s

5. استخدام واجهة الويب (Dashboard):
   sudo ./vulnscanner dashboard --port 8080
   ثم افتح المتصفح واذهب إلى: http://localhost:8080

6. توليد تقارير متقدمة:
   sudo ./vulnscanner report --format html --type detailed --output /tmp/report.html

7. استخدام الذكاء الاصطناعي للكشف عن الثغرات:
   sudo ./vulnscanner scan 192.168.1.1 --ai-detection --confidence 0.8

================================================================================
                            خيارات المسح المتقدمة
================================================================================

--ports: تحديد المنافذ (مثال: 80,443,8080 أو 1-1000)
--threads: عدد مؤشرات الترابط (الافتراضي: 50)
--timeout: مهلة الاتصال (الافتراضي: 3s)
--rate-limit: حد معدل الطلبات في الثانية
--user-agent: User-Agent مخصص
--proxy: استخدام بروكسي (مثال: http://proxy:8080)
--stealth: وضع التخفي لمنع الكشف
--ai-detection: تمكين الكشف بالذكاء الاصطناعي
--confidence: عتبة الثقة للكشف AI (0.0-1.0)

================================================================================
                            أوامر التقارير
================================================================================

1. تقرير HTML تفاعلي:
   sudo ./vulnscanner report --format html --type detailed --scan-id scan_123

2. تقرير JSON للتحليل:
   sudo ./vulnscanner report --format json --type summary --output report.json

3. تقرير CSV للجداول الحسابية:
   sudo ./vulnscanner report --format csv --type executive --output vuln_report.csv

4. تقرير PDF احترافي:
   sudo ./vulnscanner report --format pdf --type detailed --output security_audit.pdf

================================================================================
                            ميزات الواجهة الويب
================================================================================

1. لوحة التحكم في الوقت الفعلي:
   - عرض حالة المسح الحالي
   - إحصائيات مباشرة للثغرات
   - رسوم بيانية تفاعلية

2. إدارة المهام:
   - بدء/إيقاف المسح
   - جدولة المهام
   - مراجعة النتائج السابقة

3. عرض النتائج:
   - جدول الثغرات المصنفة حسب الخطورة
   - تفاصيل كل ثغرة مع التوصيات
   - تصدير النتائج بصيغ مختلفة

4. إدارة التقارير:
   - توليد تقارير مخصصة
   - حفظ التقارير السابقة
   - مشاركة النتائج

================================================================================
                            أمثلة عملية
================================================================================

مثال 1: مسح شبكة الشركة الصغيرة
sudo ./vulnscanner scan 192.168.1.0/24 --ports 1-1000 --threads 100

مثال 2: مسح سريع للخوادم الحساسة
sudo ./vulnscanner scan 10.0.0.1,10.0.0.2 --ports 22,80,443,3306,3389 --stealth

مثال 3: مسح شامل مع تقرير تنفيذي
sudo ./vulnscanner scan target.com --ai-detection --format pdf --type executive

مثال 4: مراقبة مستمرة عبر الواجهة الويب
sudo ./vulnscanner dashboard --port 8443 --ssl --auth

================================================================================
                            نصائح الأمان
================================================================================

1. استخدم VPN أو SSH tunneling عند المسح عن بعد
2. احصل على إذن كتابي قبل مسح أي نظام
3. لا تستخدم الأداة ضد الأنظمة التي لا تملكها أو لا تملك إذنًا بمسحها
4. راقب سجلات النظام أثناء المسح
5. استخدم وضع التخفي عند الضرورة
6. احفظ التقارير في مواقع آمنة

================================================================================
                            استكشاف الأخطاء وإصلاحها
================================================================================

مشكلة: "Permission denied"
الحل: استخدم sudo أو قم بتشغيل الأداة كـ root

مشكلة: "Connection timeout"
الحل: زد من قيمة --timeout أو تحقق من اتصال الشبكة

مشكلة: "Too many open files"
الحل: قلل من عدد --threads أو عدل من حدود النظام

مشكلة: التقارير لا تُنشأ
الحل: تأكد من وجود مساحة قرص كافية وصلاحيات الكتابة
